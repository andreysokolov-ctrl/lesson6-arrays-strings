# lesson6-arrays-strings
Задачи на обработку массивов и строк. Занятие 6. Вариант 25

## Задача 1: Обработка массивов
**Условие:** На заданном отрезке [a, b] вычислить 30 значений функции
f(x) = e^(-x) * sin(6x) и поместить в массив F, разделив каждый элемент
на последнее положительное значение массива.

**Файл:** `task1_arrays.cpp`

### Тесты:
| Ввод (a, b) | Ожидаемое поведение |
|---|---|
| a=0, b=3 | Массив из 30 значений, делённых на последнее положительное |
| a=1, b=1 | Ошибка: a должно быть меньше b |
| a=0, b=0.1 | Ошибка (возможно, нет положительных значений) |

---

## Задача 2: Обработка строк
**Условие:** Из текста с тегами `<strong>...</strong>` вывести ФИО героев ВОВ.
Строки с неполной разметкой игнорируются.

**Файл:** `task2_strings.cpp`

### Тесты:
| Ввод | Вывод |
|---|---|
| `<strong>Жуков Георгий Константинович</strong>` | `Жуков Георгий Константинович` |
| `<strong>Без закрывающего тега` | *(нет вывода)* |
| `Без открывающего тега</strong>` | *(нет вывода)* |
| `</strong>Неправильный порядок<strong>` | *(нет вывода)* |
